# QPE 기반 블랭크 감산 Jacobian 보정 — 분석 및 검증 보고서

> Hidex LSC의 로그 스케일 에너지 축 특성을 반영한 퀜칭 보정 알고리즘 근거 및 실험 데이터 검증
> 검증 데이터: UGAB_jeongsoo_exp01 Rep01~Rep03 (N=29)

---

## 01. 배경 및 문제 정의

### 왜 블랭크를 그대로 빼면 안 되는가

샘플과 블랭크의 **퀜칭(Quenching) 정도가 다를 경우**, 두 스펙트럼의 에너지 축 스케일이 달라진다.
이 상태에서 단순 채널별 감산을 수행하면 채널 불일치로 인한 체계적 오차가 발생한다.

| 항목 | 값 |
|---|---|
| 샘플 QPE (A01, REP03) | 796.1 (퀜칭 강함, 질산 소스) |
| 블랭크 QPE (B01, REP03) | 826.5 (퀜칭 약함, 순수 칵테일) |
| QPE 차이 | 30.4 |
| 에너지 비율 k | 0.8233 = E(샘플) / E(블랭크) |
| 샘플 에너지 | 472.4 MeV |
| 블랭크 에너지 | 573.8 MeV |

### Hidex 채널-에너지 변환의 비선형성

```
E(ch) = 10^((ch+173)/362) − 3  [MeV]
∴ dE/dch ∝ (E+3)  // 채널당 에너지 폭이 에너지에 비례
```

고채널로 갈수록 동일한 Δch가 더 큰 ΔE에 대응하므로, **단순 채널 Shift는 물리적으로 부정확**하다.
퀜칭 차이에 의한 에너지 축 변화는 **에너지 공간에서의 Stretch**로 모델링해야 한다.

## 02. 실험적 근거 — QPE vs 고에너지 피크 위치 상관관계

Rep01~Rep03 데이터(29 포인트, REP02 A03 이상치 제외)에서 870~970ch 구간 고에너지 노이즈 피크 위치를 Savitzky-Golay 평활화 후 추출, QPE와 선형 회귀 수행.

### 회귀 분석 결과

```
Peak_CH = 0.6906 × QPE + 374.93   (R² = 0.9358)
// QPE 1 단위 변화 → 고에너지 피크 0.69ch 이동
```

| 지표 | 값 |
|---|---|
| 기울기 | 0.6906 ch/QPE |
| 절편 | 374.93 ch |
| R² | **0.9358** (매우 강한 상관) |
| 데이터 포인트 | 29 |

> **결론**: QPE 값의 변화가 스펙트럼 전체 에너지 축의 이동량과 직결됨이 실험적으로 입증됨.
> 샘플-블랭크 간 QPE 차이를 에너지 비율 k로 환산하여 블랭크 스펙트럼을 보정하는 접근이 타당하다.

---

## 03. 보정 알고리즘 — 3단계 절차

### Step A: 물리적 보정 계수 k 도출
```
E(q) = 10^((q+173)/362) − 3
k = E(QPE_sample) / E(QPE_blank)
k = 472.41 / 573.83 = 0.8233
```

### Step B: 에너지 역매핑 (Inverse Mapping)
```
E_src = E_target / k
ch_src = 362 · log₁₀(E_src+3) − 173
→ np.interp(ch_src, ch, blank)
```

### Step C: Jacobian 밀도 보정
```
J(E) = (E+3) / (E+3k)
Count_corr = Count_stretched × J
보존율: 99.99%
```

**Jacobian 의미**: k < 1 (샘플 QPE < 블랭크 QPE) 일 때, 스트레칭 시 저에너지 bin이 좁아져 계수 밀도가 증가한다. J < 1이므로 과계상분을 제거한다.

- 저채널(E 작음): J ≈ 1/k → 강한 보정
- 고채널(E 큼): J ≈ 1 → 보정 미미

## 04. 실험 데이터 검증

### 단순 감산 vs QPE-Jacobian 보정 감산 비교 (0~300ch)

| 방법 | 계수 합계 | 차이 |
|---|---|---|
| 단순 감산 | 1,959 counts | — |
| QPE-Jacobian 보정 | 1,821 counts | **−138 (−7.0%)** |

QPE 보정 적용 시 저에너지 잔여 계수가 7.0% 감소 → 단순 감산 시의 과소 차감 오차가 보정됨.

### 단순 채널 Shift vs 에너지 Stretch 비교

| 방법 | ch=300 이동량 | ch=700 이동량 | 물리적 타당성 |
|---|---|---|---|
| 단순 Shift (QPE 차이=30ch) | −30ch (고정) | −30ch (고정) | ❌ 로그 스케일 무시 |
| 에너지 Stretch (k=0.8233) | −25.7ch | −30.2ch | ✅ 로그 스케일 반영 |

에너지 Stretch는 채널마다 이동량이 다르다 — 저채널에서 이동이 작고 고채널로 갈수록 커진다.
