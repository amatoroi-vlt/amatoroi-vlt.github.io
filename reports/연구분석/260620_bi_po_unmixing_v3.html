Bi-210 / Po-210 스펙트럼 분리 분석 — Stage 1 (v3)

분석 대상: 260620_project_15rep.lsc  |  작성일: 2026-06-24  |  방법: 시계열 최소제곱 역산 + CPM 성분 분해 검증 + d²S/dc² 진단
요약
Bi-210과 Po-210이 미지 비율로 혼합된 시료(A02, 15회 반복) 및 블랭크(A01, 5회)를 대상으로 2-기저 최소제곱 역산을 수행하였다. 전체 피팅 R² = 0.9942–0.9995로 수학적 적합도는 양호하나, 역산된 Bi-210 스펙트럼의 d²S/dc² 분석에서 7개의 음수 극솟값이 확인되었다. 순수 beta 방출체(Bi-210)의 단봉 스펙트럼 조건과의 비교를 통해 현재 분리의 물리적 타당성을 정량 평가하였다. Po-210 제거 후 Bi-210 추출 CPM의 자유 λ 피팅 결과 T½ = 4.986일 (기대값 5.013일, 편차 +0.53%)이다.

1.실험 배경 및 분석 목적

Bi-210(T½ = 5.013일)과 Po-210(T½ = 138.376일)이 미지 비율로 혼합된 시료를 8.70일에 걸쳐 반복 계측함으로써, 두 핵종의 반감기 차이에서 비롯되는 스펙트럼 시간 변화를 역산 정보원으로 활용한다. alpha/beta 분리 없이 beta_triple + alpha_triple을 합산한 단일 1024채널 스펙트럼을 사용하며, 분리는 순전히 시계열 붕괴 동역학에 의존한다.

2.물리 모델 및 분석 방법

2.1 완전 Bateman 방정식과 2-기저 모델

시각 t에서의 블랭크 차감 스펙트럼은 다음과 같이 표현된다:

$$S_{\text{net}}(c,\,t) = f_{\text{Bi}}(c)A_{\text{Bi}}(0)e^{-\lambda_{\text{Bi}}t} \;+\; f_{\text{Po}}(c)\!\left[A_{\text{Po}}(0)e^{-\lambda_{\text{Po}}t} +A_{\text{Bi}}(0)\kappa\!\left(e^{-\lambda_{\text{Bi}}t}-e^{-\lambda_{\text{Po}}t}\right)\right]$$

Bateman 성장항(κ = -0.0376)을 재배열하면 두 기저 함수의 선형결합으로 정리되며, 최소제곱이 역산하는 겉보기 계수 xBi(c), xPo(c)는 순수 스펙트럼에 κ 항이 혼재된 값이다 (|κ| ≈ 3.8%). 기저 행렬 Φ(15×2) 조건수 = 6.61.

2.2 CPM 기반 겉보기 혼합비 추정

Po-210이 관측 구간 내 거의 상수(9일간 변화 4.3%)이므로, 총 net CPM에 다음 모델을 피팅하면 겉보기 혼합비를 독립적으로 추정할 수 있다:

$$\text{CPM}_{\text{net}}(t) = B\cdot e^{-\lambda_{\text{Bi}}t} + C$$

여기서 C는 Po-210의 CPM 기여(검출 효율 포함)이다. 단, 이 비율은 진정한 활동도 비가 아니라 검출 효율비 kPo/kBi가 포함된 겉보기 값임을 명시한다. 피팅 결과: B = 213.4 CPM, C = 79.2 CPM, 겉보기 비율 C/B = 0.371.

6.역산 스펙트럼 물리적 적합성 진단

6.1 d²S/dc² 분석 — 피크 구조 검증

순수 beta 방출체 Bi-210의 스펙트럼은 단봉(unimodal)이어야 하며, d²S/dc²는 정확히 하나의 음수 극솟값만 가져야 한다. 2차 미분이 두 번 이상 음수가 되면 이중 피크(double peak)를 의미하며 물리적으로 비타당하다.

6.2 d²S/dc² 이상 징후의 물리적 해석

잘못된 Bi-210 역산 스펙트럼에서 d²S/dc²에 나타나서는 안 될 패턴을 정리한다:

  1. ch ~580 근방의 2차 음수 극솟값: Po-210이 완전히 분리되지 않고 Bi-210 성분에 잔류하는 경우. 역산이 "시간에 따라 변하는 성분"으로 일부 Po-210 스펙트럼을 흡수한 결과.
  2. Emax 해당 채널 이상에서 d²S/dc² < 0: Bi-210의 최대 beta 에너지(1.162 MeV) 상한보다 높은 채널에서 스펙트럼이 올라가면 에너지 상한 위반.
  3. 음수 계수 채널 주변의 급격한 부호 전환: 스펙트럼이 0 아래로 내려가는 구간에서 d²S/dc²가 큰 양수 스파이크 → 비물리적 역전.

7.CPM 성분 분해 검증 — 자유 λ 피팅

Po-210 기여(xPo·exp(−λPo·t))를 각 시점 net 스펙트럼에서 차감하면 순수 Bi-210 기여 CPM 시계열을 얻는다. 이 시계열에 λ를 자유 파라미터로 지수 피팅하여 모델 내적 일관성을 검증한다.

⚠️ 수학적 참고: 이 검증은 완전히 독립적이지 않다 — 추출 CPM이 exactly spec_bi.sum()·exp(−λBi·t)/300 에 잔차 합산을 더한 값이므로, 잔차가 작고 무작위적이면 λ_fit ≈ λBi가 항상 성립한다. 의미 있는 편차가 나타날 경우 잔차에 λBi와 다른 주파수의 계통 성분이 있음을 시사한다.

9.모델 타당성 검토

9.1 초기 혼합비 미지 문제

겉보기 비율 C/B = 0.371는 CPM 단위의 비율이며, 활동도 비 APo/ABi로 변환하기 위해서는 검출 효율비 kPo/kBi가 필요하다. Bi-210은 beta 방출체, Po-210은 alpha 방출체이므로 TDCR 방식에서 두 효율이 동일하다고 가정할 수 없다.

9.2 Po-210 기여의 상수 근사 타당성

9일간 Po-210 감쇠율은 4.26%이다. 이보다 큰 잔차나 CPM 추세가 관찰될 경우 추가 성분(예: Bi-210 이외의 beta 방출체) 또는 블랭크 시간 불일치를 의심해야 한다.

9.3 2성분 모델의 한계

d²S/dc² 분석에서 확인된 다중 피크 패턴은 2성분 모델이 데이터를 완전히 설명하지 못할 가능성을 제기한다. 잔차 스펙트럼의 구조적 패턴이 통계적 노이즈 수준인지, 추가 성분인지를 구분하기 위한 추가 분석이 필요하다.

10.최종 결론

결론 1 — 수학적 분리 성공
R² > 0.994로 2-기저 최소제곱이 시계열 데이터를 잘 설명한다. CPM 분해(C/B = 0.371)와 스펙트럼 역산(x_Po/x_Bi = 0.403)이 편차 7.9%로 정합적이다.
결론 2 — 물리적 타당성 (❌ 다중 피크 (7개, ch380, ch398, ch468, ch490, ch562, ch610, ch647) — 단봉 beta 스펙트럼 위반)
역산 Bi-210 스펙트럼의 d²S/dc²에서 7개의 음수 극솟값이 확인되었다. 단봉 beta 스펙트럼의 물리적 조건(극솟값 = 1)과 불일치한다. 이는 Po-210 스펙트럼과의 채널 공간 중첩 (20채널 차이)으로 인해 2-기저 모델이 두 성분을 완전히 분리하지 못하였을 가능성을 시사한다.
